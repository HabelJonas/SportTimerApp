# Sport Timer

[![CI](https://github.com/HabelJonas/SportTimerApp/actions/workflows/ci.yml/badge.svg)](https://github.com/HabelJonas/SportTimerApp/actions/workflows/ci.yml)

Es handelt sich hierbei um eine Kotlin Android App zum timen von Sport Sessions.

## ✨ Features

- Workout Profile anlegen/editieren
- Timer läuft im Hintergrund

## 🧱 Tech Stack

- **Programmiersprache:** Kotlin
- **Build tooling:** Gradle
- **CI/CD:** GitHub Actions

## 🗺️ Roadmap (Ideas)

- Import/Export von Trainingsprofilen
- UI/UX verbessern
- Verschiedene Sounds anbieten für Phasenwechsel
- In den Google Play Store ausrollen
- In Sportgruppen und Foren bewerben
